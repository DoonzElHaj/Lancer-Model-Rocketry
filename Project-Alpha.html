Project Alpha

Project Alpha - LB-1 - Mark 1

Description: Project Alpha is the first project in the Nova series, designed as a introduction for beginners for the basics of rocketry. This low-budget project focuses on simple construction and safe launches.

Note: If a upgrade is made to a rocket, it will be documented under a new rocket generation. (Mk1, mk2, etc.)

Objectives:

  • Build a rocket to reach an altitude of +50 meters.
  • Test basic aerodynamic properties
  • Understand and troubleshoot operational issues

Status: Currently not in Development

Estimated Completion: To Be Determined

Team: Open to all members
