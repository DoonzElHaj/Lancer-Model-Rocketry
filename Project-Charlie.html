Project Charlie

Project Charlie - LB-3 - Mark 1

Description: Project Charlie is the third project in the Nova series, designed as an advanced upgrade to Project Beta's LB-2, aimed to test new rocket mixtures. This medium-budget project will advance our mixture testing capabilities.

Note: If a upgrade is made to a rocket, it will be documented under a new rocket generation. (Mk1, mk2, etc.)

Objectives:

  • Build a rocket to reach an altitude of +100 meters.
  • Test new rocket mixtures
  • Understand and troubleshoot operational issues

Status: Currently not in Development

Estimated Completion: To Be Determined

Team: Open to all members
