Hello, this readme will give a short summary of why I am doing this and an explanation of each file.

Why: The same reason as you( I hope ), because planes are interesting and have lots of capabilities. Note that this site originally started as a hobby project for HTML( I'm interested in coding and planes ).

index.html: The home page of Lancer Model Aviation, GitHub(GH) searches for an index.html file as a "homeroom" from what I understand.

Lancer-Contacts.html: The contact page for the lead devs and soon anyone else working on this project. (Feel free to join!)

Lancer-Projects.html: Showcases all projects.

Lancer-Upcoming.html: Showcases future plans for planes and how we intend to build them.

Lancer-Units.html: A unit "libary" were all electronics use will be here.

Lr-1.html, Lr-2.html, Lr-3.html: Different projects, ideas and specs for the Lr- plane series, aswell as objectives and what not.

LSu-1_Mk1.ino: Controls the servos attached to elevators and aileron. The LSu-1 will be made of an Arduino R3, the MPU6050 to measure the aircraft's angle and finaly servos connected elevators and potentially the aileron.


Note that the orginal name for this project was "Nova", now changed to Lancer.
